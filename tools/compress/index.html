メディアを圧縮（画像・動画・音声）

画像・動画・音声はサーバーに送信されません。すべてこの端末（ブラウザ）内で処理されます。

ここに画像をドラッグ＆ドロップ
またはクリックしてファイルを選択（複数選択可）

数値が大きいほど高画質・容量は大きくなります。

アスペクト比は維持されます。空欄の場合は原寸のまま処理します。

    ここに動画をドラッグ＆ドロップ
    またはクリックしてファイルを選択（1ファイル）

    出力形式: MP4（H.264 / libx264 + AAC音声）。 アスペクト比は維持されます。解像度を下げると容量削減効果が高まります。

      動画圧縮は端末内のffmpeg.wasm（単一スレッド）で行うため、 ファイルが大きいほど処理に時間がかかり、メモリも多く使用します。 数十MB以下・短めの動画を推奨します。長尺・高解像度はブラウザが不安定になる場合があります。

      ここに音声をドラッグ＆ドロップ
      またはクリックしてファイルを選択（1ファイル）

      出力形式: MP3（libmp3lame）。 ビットレートを下げるほど容量は小さく、音質はわずかに低下します。

        音声圧縮は端末内のffmpeg.wasm（単一スレッド）で行います。 長時間の音声はメモリ・処理時間の制約を受けます。

        使い方

        1. 上部のタブで「画像 / 動画 / 音声」モードを選びます。
        2. 圧縮したいファイルをドラッグ＆ドロップ、またはボタンから選択します（画像は複数枚OK）。
        3. 必要に応じて画質・解像度・ビットレートなどを調整します。
        4. 処理が完了すると、元のサイズ・圧縮後のサイズ・削減率が表示されます。
        5. 個別にダウンロード、または「すべてダウンロード」で保存できます。

        画像はCanvas API、動画・音声はブラウザ内で動作するffmpeg.wasm（単一スレッド版）で処理され、 ファイルが外部のサーバーへ送信されることはありません。 保存されるのは設定情報のみで、メディアデータは含まれません。 動画・音声モードは初回実行時に圧縮エンジン（約25MB）を端末内に読み込みます。

        よくある質問

        画像はアップロードされますか？

        いいえ。本ツールは画像データを一切サーバーに送信しません。 選択した画像はブラウザ内のメモリ上でのみ処理され、処理が終わるとページを閉じることで消去されます。

        画質はどれくらい下がりますか？

        JPEG/WebPは非可逆圧縮のため、画質スライダーの値を下げるほど画質はわずかに低下し、 その分ファイルサイズは小さくなります。「画質を保ちながら容量を大きく削減」できる バランスの良い設定を探すのにスライダーをご活用ください。 なお、PNG形式は可逆圧縮のため、圧縮による容量削減効果は小さくなります。

        対応している画像形式は？

        入力はJPEG・PNG・WebP・GIFなど、ブラウザが対応する一般的な画像形式に対応しています。 出力はJPEG・WebP・PNGから選択できます（「自動」を選ぶとJPEGで圧縮されます）。

        動画・音声はどの形式で出力されますか？

        動画はMP4（H.264 / libx264、音声はAAC）で出力します。音声はMP3（libmp3lame）で出力します。 いずれも広く再生できる形式です。入力は一般的な動画・音声形式に対応します （端末内のffmpeg.wasmがデコードできる範囲）。

        動画・音声もアップロードされませんか？

        されません。動画・音声の圧縮は、リポジトリに同梱したffmpeg.wasm（単一スレッド版）を用いて ブラウザ内だけで実行します。外部のサーバーやCDNへファイルを送信することはありません。 圧縮エンジン（約25MB）は動画・音声モードでの初回実行時に読み込まれ、以降はキャッシュされます。

        大きい動画でも処理できますか？

        原理的には可能ですが、ブラウザ内処理のため、ファイルが大きいほど処理時間が長くなり メモリ使用量も増えます。数十MB以下・短めの動画を推奨します。 長尺・高解像度の動画はブラウザが不安定になる場合があります。

        写真の向きが回転してしまうことはありますか？

        スマートフォン等で撮影した画像のEXIF回転情報を読み取り、正しい向きで処理するように 実装しています。一部の古いブラウザでは向きの自動補正が効かない場合があります。

        一度に何枚まで処理できますか？

        枚数の制限は設けていませんが、画像は1枚ずつ順番に処理されます。 大量・大容量の画像を選択すると、処理に時間がかかる場合があります。
